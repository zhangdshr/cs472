1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent.
All of this fragment codes are equivalent, depending our needs we can use them, since the each method accepts two parameter, we can use the method either by passing all the two parameter or without passing any parameter by using this keyword to refer the element which we want to iterate.
//code fragment 1
$("li").each(function(idx, e) {
$(e).css(“color”, “yellow”); });
The second fragment shows that it’s not accepting any parameter, but it’s using this keyword to change the color of the element.
//code fragment 2
$("li").each(function() {
$(this).css(“color”, “yellow”); });
The third fragment shows that, it is accepting the index parameter but it’s not applicable at this point.
//code fragment 3
$("li").each(function(idx) {
$(this).css(“color”, “yellow”); });
